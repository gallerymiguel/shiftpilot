# ShiftPilot

ShiftPilot is an AI-powered workforce disruption assistant designed to help managers respond to call-outs, staffing gaps, and schedule disruptions.

## Planned Stack

- FastAPI
- Postgres
- n8n
- OpenAI API
- Chroma
- Docker

## MVP Goal

Receive a simulated employee text, detect a scheduling disruption, find replacement candidates, and send a manager-ready recommendation.
